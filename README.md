# Code Cartographer Redux\n\nOptimized knowledge graph generation tools powered by Gemini 3.0 Pro.\n\n## Contents\n\n*   **repomix.config.json**: Optimized configuration for Repomix to filter noise upstream.\n*   **graph_generation_prompt.md**: The 'Prompt as Code' driver for generating Dependency and Class diagrams.\n*   **generate_graph.sh**: Automation script to run the workflow.\n\n## Usage\n\n1.  Install Repomix: `npm install -g repomix`\n2.  Run the script: `./generate_graph.sh`
